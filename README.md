# Sudoku-solver

This is an machine learning based end-to-end application that help solve sudoku puzzle. The model is built using CNN architecture and has achieved an accuracy of 98%. The UI of the model is made using Flask and HTML. The CNN architecture is made using Tensorflow from the scratch and has multiple hidden layers to increase the efficiency. 

The user need to manually enter the digits into the sudoku board and it gives out the complete board after the submit button is hit.
